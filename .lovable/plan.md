

## Change Accent Color to #E7C8CD

The hex `#E7C8CD` converts to approximately `HSL(350, 39%, 84%)`.

### Changes

**`src/index.css`** — Update the `--accent` CSS variable in the `:root` (light mode) block:
- Change `--accent: 350 33% 85%;` → `--accent: 350 39% 84%;`

This will affect all elements using the `accent` color token across the site (headings labels, contact button, etc.). Dark mode accent remains unchanged.
